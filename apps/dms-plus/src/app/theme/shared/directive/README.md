# DMS Plus - Shared Directives

This directory contains custom Angular directives used throughout the DMS Plus application. These directives provide reusable functionality and enhance the user interface components.

## Custom Directives:

- **Form Validation Directives**: Custom validation for Pakistani business rules
- **Number Formatting Directives**: PKR currency and numeric formatting
- **Permission Directives**: Role-based element visibility control
- **Auto-focus Directives**: Enhanced user experience for form navigation
- **Loading State Directives**: Loading indicators for async operations
- **Data Format Directives**: Date and time formatting for local business context

These directives help maintain consistency across the application and provide specialized functionality for distribution management workflows.
