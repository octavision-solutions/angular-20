# DMS Plus - Shared Filters & Pipes

This directory contains custom Angular pipes and filters used throughout the DMS Plus application. These utilities provide data transformation and formatting capabilities.

## Custom Pipes & Filters:

- **Currency Pipe**: PKR currency formatting for Pakistani business context
- **Date Pipes**: Local date and time formatting with Urdu calendar support
- **Search Filters**: Advanced search and filtering for business data
- **Number Formatting**: Quantity, percentage, and numeric display pipes
- **Status Filters**: Business status and categorization filters
- **Export Filters**: Data preparation for reports and exports

These pipes enhance the user experience by providing properly formatted data display and efficient filtering capabilities for distribution management operations.
