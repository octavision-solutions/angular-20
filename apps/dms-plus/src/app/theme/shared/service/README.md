# DMS Plus - Shared Services

This directory contains shared services used throughout the DMS Plus application. These services provide core functionality and business logic that can be accessed by multiple components.

## Shared Services:

- **Database Service**: Core data operations and IndexedDB management
- **Authentication Service**: User authentication and session management
- **Sync Service**: Offline-to-online data synchronization
- **Utility Services**: Common helper functions and utilities
- **Validation Services**: Form and data validation logic
- **Export Services**: Data export and reporting functionality

These services follow Angular's dependency injection pattern and are designed for reusability and maintainability across the entire application.
