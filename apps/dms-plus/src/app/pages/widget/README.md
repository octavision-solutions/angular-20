# DMS Plus - Dashboard Widgets

This directory contains widget components for the DMS Plus dashboard. These widgets provide key business insights and metrics for Pakistani distribution businesses.

## Available Widgets:

- **Sales Summary**: Daily, weekly, and monthly sales performance
- **Inventory Status**: Stock levels and low stock alerts
- **Cash Flow**: Real-time cash position and flow analysis
- **Customer Analytics**: Top customers and payment status
- **Route Performance**: Sales performance by geographic routes
- **Expense Tracking**: Recent expenses and category breakdowns

The widget system is designed to provide at-a-glance business intelligence for quick decision making in distribution operations.
