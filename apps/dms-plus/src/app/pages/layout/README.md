# DMS Plus - Layout Components

This directory contains layout components and templates for the DMS Plus application. These components provide consistent user interface structure and design patterns throughout the system.

## Layout Components:

- **Admin Layout**: Main application layout with navigation and content areas
- **Guest Layout**: Public pages layout for authentication and landing pages
- **Navigation Components**: Sidebar navigation, breadcrumbs, and menu structures
- **Header Components**: Top navigation bar and user profile area
- **Footer Components**: Application footer with company branding and links

The layout system ensures responsive design and consistent user experience across all devices and screen sizes, optimized for Pakistani business workflows.
